🚨 COMMON ISSUES & SOLUTIONS
=============================

ISSUE: Claude not recognizing file structure
SOLUTION: Ensure first row has clear column headers, no empty rows

ISSUE: Date format errors causing matching problems
SOLUTION: Standardize all dates to DD-MM-YYYY format before upload

ISSUE: GSTIN validation errors
SOLUTION: Check GSTIN is exactly 15 characters with no spaces

ISSUE: Charts not displaying correctly in Excel
SOLUTION: Open file in Excel (not Google Sheets), enable macros if prompted

ISSUE: Formulas showing as #REF! errors
SOLUTION: Recalculate workbook (Ctrl+Alt+F9 in Excel)

ISSUE: Processing taking too long (>10 minutes)
SOLUTION: Split large files (>2000 rows) into smaller batches

ISSUE: Dashboard colors not matching company brand
SOLUTION: Easy to customize after generation - just change Excel theme

ISSUE: Missing some data for certain modules
SOLUTION: System works with partial data - run available modules first
